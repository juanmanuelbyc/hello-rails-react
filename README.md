### Prerequisites

In order to run this project you need:

```

 - Laptop/Desktop.
 - Internet.
 - Ruby.
 - Rails.
 - Vs Code.

```

### Setup

Clone this repository to your desired folder:

```
 - mkdir my-folder
 - cd my-folder
 - git clone git@github.com:SaboorSohaib/Blog-app.git
 - cd Blog-app
 - bundle install
 - rails db:create
 - rails db:migrate
 - rails db:seed


 Note: For adding new greetings open the Rails console and use the command below.
 'Greeting.create(message: "New greeting text")'

```
